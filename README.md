# Owner-System-Aoi.js
An owner system that's quick and easy to set up.

# Information :
- Only server owners can add users.
- This is not a bot owner system, but a permission owner system for the servers concerned.

# Required
```js
- npm i aoi.js@6.4.0 aoi.parser
- Aoi.js Handler
```

# Variables
```js
owner: "",
owner_count: 0,
owner_page: 0,
global: true
```
